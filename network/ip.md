# ip

### Install

For the record, the following Ubuntu packages can help debug network and OS issues:
- dnsutils (nslookup)
- net-tools (netstat, ifconfig, etc.)
- iproute2 (ip)
- iputils-ping (ping)
- iputils-tracepath (traceroute6)
- proccps (ps)

```bash
sudo apt-get install iproute2
```

### Get IP address

```bash
ip addr
```
